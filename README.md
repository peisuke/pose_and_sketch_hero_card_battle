# POSE & SKETCH HERO CARD BATTLE

このゲームは、あなたの身の回りにある『日常』を、AIの力で一瞬にして『ファンタジーの伝説』へといざなうリアルタイム・バトルゲームです。Gemini の高度な画像認識と生成能力を組み合わせ、現実世界とゲームの世界の境界を溶かします。

特徴：AIが「画像」を読み取り、「文脈」を理解し、新しい「ビジュアル」を生成する。この複雑なステップを、魔法陣が回転し、データが収束していく視覚演出に変換しました。
現実世界にいる人のポーズや描いたスケッチから、バトルカードを召喚し、オンラインで対戦できるゲームです。

カメラに映る自分の手や部屋の上に魔法陣が重なることで、プレイヤー自身が「召喚師」になったかのような没入感を生み出します。

## how to use

```
uvicorn main:app --host 0.0.0.0 --port 8002
```
